## Setup

Requires environment variables, which may be provided in a `.env` file in
project root folder:

- `NODE_ENV`
- `GOOGLE_CLIENT_ID`
- `GOOGLE_CLIENT_SECRET`
- `OAUTH_CALLBACK_URI`
